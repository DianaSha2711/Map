Домашнее задание: "Контейнеры - Map"

Реализация класса ErrorRepository для централизованного хранения ошибок с использованием Map.

Требования:

  • Хранение ошибок в Map (ключ - код ошибки, значение - описание)
  • Метод translate(code) для получения описания ошибки
  • Возврат "Unknown error" для неизвестных кодов
  • 100% покрытие тестами
